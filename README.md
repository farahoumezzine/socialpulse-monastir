# socialpulse-monastir
Build a near real-time social media monitoring system that tracks public sentiment and emerging topics from noisy, multilingual social media posts, Detects local events Generates prioritized alerts for local stakeholders

## Langues prises en charge
- Français (FR)
- Arabe standard (AR)
- Darija tunisien (DA)

## Structure
- `/data` : jeux de données
- `/src` : code source
- `/notebooks` : expérimentations
- `/docs` : rapport, éthique

## ⚠️ Security Note

API credentials are managed via environment variables and are **not included** in this repository.
To run scripts that require the X API, create a `.env` file in the project root:
